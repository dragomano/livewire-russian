# Директива `wire:init`

Livewire предлагает директиву `wire:init` для выполнения действия сразу после рендеринга компонента. Это может быть полезно в случаях, когда вы не хотите задерживать загрузку всей страницы, но хотите загрузить некоторые данные сразу после загрузки страницы.

```html
<div wire:init="loadPosts">
    <!-- ... -->
</div>
```

Действие `loadPosts` будет выполнено немедленно после рендеринга компонента Livewire на странице.

Однако в большинстве случаев [функция отложенной загрузки Livewire](/features/lazy) предпочтительнее использования `wire:init`.

## Справочник

```php
wire:init="действие"
```

Эта директива не имеет модификаторов.
